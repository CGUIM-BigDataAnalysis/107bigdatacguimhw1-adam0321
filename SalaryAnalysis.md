### 106年度薪資較103年度薪資高的職業有哪些?

``` r
#這是R Code Chunk
#將103和106做合併，並叫job
job <- merge(X103teach, X106teach, by = "大職業別")
#將106年的大學畢業薪資除以103大學畢業薪資，並存入job$jobrate欄位
job$jobrate <- c(job$`大學-薪資.y` / job$`大學-薪資.x`)
#用jobrate將job做大到小的篩選
job <- arrange(job,
               desc(jobrate))
#將jobratejob>1在存入job1中
job1 <- subset(job, job$jobrate > 1)
```
